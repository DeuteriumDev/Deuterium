# dt_cli

## Environment variables

This package needs the following variables in a .env file:

```bash
DEBUG=true # only needed for development purposes
DB_CONNECTION=postgresql://<USER_NAME>:<PASSWORD>@<SERVER_ADDR>:<PORT?>/<SCHEMA>

```
